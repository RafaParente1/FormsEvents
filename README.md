# Formulário de Eventos/Stands

Projeto desenvolvido para simplificar a organização e a produção de feiras, eventos corporativos e stands. A plataforma funciona como um checklist interativo e inteligente, permitindo o acompanhamento detalhado de todas as etapas — do pré-evento ao dia da realização —, além de oferecer recursos para exportação e importação de dados através de arquivos.json.
